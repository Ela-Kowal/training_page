It is better to use "strong" tag when we want to make text bold. "B" tag only makes our text look darker than the rest and it doasn't contain any meaning. Strong tag informes browser, that this text has strong importance, so it looks diferent. The same thing is when we want to italize text. We should use "em" tag instead of "i" tag. Both of them make the text apear te same way, but only "em" tag informes the browser about importance of our text.
